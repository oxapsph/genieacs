# ACS-Ubuntu-22.04
This is autoinstall GenieACS For ubuntu version 22.04 (Jammy)

# Usage
```
sudo su
```
```
wget https://github.com/oxapsph/genieacs/GACS-Jammy.sh
```
```
chmod +x GACS-Jammy.sh
```
```
sudo apt-get install dos2unix
```
```
dos2unix GACS-Jammy.sh
```
```
bash GACS-Jammy.sh
```
